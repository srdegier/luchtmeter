## Connecties

### SGP30 sensor

 * GND on GND
 * VCC on 3.3V
 * SCL on esp8266 D1
 * SDA on esp8266 D2

### DHT11/DHT22 sensor

 * GND on GND
 * VCC on 3.3V
 * sensor to D5 / GPIO14
